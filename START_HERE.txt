Welcome to BrazaReader - A lightweight PDF reader for Raspberry Pi Zero!

Before using the software read the terms on TERMS.docx.

This project contains everything you need to build and run a PDF reader
optimized for ARMv6 devices with minimal hardware resources.

================================================================================
QUICK START (5 MINUTES)
================================================================================

1. READ THIS FIRST:
   → Open: QUICKSTART.md

2. BUILD:
   → Run: chmod +x build.sh && ./build.sh

3. SETUP:
   → Create: mkdir -p /library ~/.local/share/brazareader

4. ADD PDFS:
   → Copy your PDF files to: /library/

5. RUN:
   → Execute: cd build && ./bin/brazareader

================================================================================
SYSTEM REQUIREMENTS
================================================================================

Desktop (to build):
  - Linux (Ubuntu/Debian)
  - CMake 3.16+
  - Qt5 development libraries
  - MuPDF development libraries
  - SQLite3
  - G++ with C++17 support

Runtime (Raspberry Pi):
  - Raspberry Pi Zero
  - ARMv6 hard-float compatible
  - 512MB RAM minimum
  - 100MB storage
  - Touchscreen (optional but recommended)

GETTING STARTED
================================================================================

STEP 1: Read Documentation
  └─ Start with: QUICKSTART.md (5 minutes)

STEP 2: Install Dependencies
  └─ On Ubuntu: sudo apt-get install cmake qt5-qmake qtbase5-dev \
                  libmupdf-dev libsqlite3-dev

STEP 3: Build
  └─ Run: chmod +x build.sh && ./build.sh

STEP 4: Setup
  └─ Create: mkdir -p /library ~/.local/share/brazareader

STEP 5: Use
  └─ Run: cd build && ./bin/brazareader

STEP 6: Add PDFs
  └─ Copy PDF files to: /library/

================================================================================
FOR RASPBERRY PI ZERO
================================================================================

See BUILD.md section "Cross-Compilation for Raspberry Pi Zero" for:
  • Setting up cross-compilation toolchain
  • Building for ARMv6 architecture
  • Deploying to physical device

Quick reference:
  1. Set up ARM sysroot
  2. Run: ./cross-compile-rpi.sh
  3. Transfer binary: scp build-rpi/bin/brazareader pi@raspberrypi:/usr/local/bin/
  4. SSH and run on Pi

USEFUL COMMANDS
================================================================================

# Compile for desktop
./build.sh

# Compile for Raspberry Pi
export SYSROOT=/path/to/sysroot
./cross-compile-rpi.sh

# Create required directories
mkdir -p /library ~/.local/share/brazareader

# Run the application
cd build
./bin/brazareader

# Check dependencies
pkg-config --exists Qt5Core && echo "Qt5 OK" || echo "Qt5 Missing"

# View settings
cat ~/.local/share/brazareader/settings.ini

# View reading history
sqlite3 ~/.local/share/brazareader/progress.db ".tables"
